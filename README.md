# Hao Wei — Personal Academic Website

A clean, single-page academic website for myself[Hao Wei](https://howw-way.github.io/personal-web/). 

## Overview

This site showcases my research, publications, and academic background. It is built as a lightweight static site using plain HTML, CSS, and JavaScript — no build tools or frameworks required.

## Structure

```
website/
├── index.html              # Main page (single-page layout)
├── projects.json           # Publication data (also inlined in index.html)
├── assets/
│   ├── css/style.css       # Custom styles
│   ├── images/             # Publication thumbnails & avatar
│   └── uploads/            # PDFs (papers, CV)
└── README.md
```

## Sections

- **About** — Bio, research interests, and social links
- **News** — Recent academic milestones
- **Publications** — Full list with PDF, arXiv, code, and project page links
- **Experience & Education** — Timeline of positions and degrees
- **Contact** — Email, GitHub, and office address

## Tech Stack

- **HTML5** + **Bootstrap 4** for layout
- **Vanilla CSS** with custom design tokens
- **Barlow** (Google Fonts) for typography
- **Font Awesome 4** for icons
- No build step — just open `index.html` or deploy to any static host

## Deployment

This site can be deployed with [GitHub Pages](https://pages.github.com/):

1. Go to **Settings → Pages** in this repository
2. Set the source to the `main` branch and root (`/`)
3. The site will be live at `https://howw-way.github.io/personal-web/`
